# JLPTDeck — 공격 응답 v3

> 2026-05-06. ATTACK_v3.md 10개 항목 처리 (3.9, 4.9 WITHDRAW 제외).
> 분류: CONCEDE / HOLD / REFRAME / ESCALATE.

## 0. v2 입장 변화 요약

마사키 v3의 핵심 무기 2가지:

1. **"가능하다고 인정했으면 v1.0에서 못 할 이유가 없다"** — 비용 0 / 자동 추출 가능 / 인프라 존재를 인정한 항목은 defer 근거가 약함 (1.3 TTS, 2.5/4.3 발음 함정, 3.6 SRSQuality).
2. **"비교 대상 / 측정 절차 / 안내 기준 부재"** — 측정/A/B/위젯 기준을 만들고도 회수 절차가 없으면 의사결정 불가 (1.11, 5.5, 5.12).

이로 인해 v2 REFRAME/HOLD의 6/10이 v1.0 격상으로 이동. 진짜 평행선이던 5.2도 마사키 카운터가 강해 CONCEDE.
