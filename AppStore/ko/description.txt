JLPTDeck은 일본어 단어를 한국어 뜻으로 인식하는 4지선다 단어장입니다.

JLPT N4~N1 7,316단어를 SM-2 간격 반복(SRS) 방식으로 학습합니다.
계정 가입, 클라우드 동기화, 외부 분석 SDK 모두 사용하지 않습니다 — 학습 기록은 기기 안에만 저장됩니다.

주요 기능
• 한 탭으로 시작 — 첫 실행도 홈 화면, 가입 절차 없음
• 한국어 뜻 4지선다 — 한자/히라가나 단어를 한국어 뜻으로 인식
• SM-2 SRS — 정답/오답에 따라 다음 복습 간격 자동 조정
• 같은 세션 즉시 재출제 — 오답은 큐 뒤에 다시 등장 (별도 SRS 손해 없음)
• "틀린 단어" 탭 — 누적 오답을 모아 집중 복습
• 정답 공개 후 표제어 발음 듣기 (TTS, 무음 모드 존중)
• 발음 함정 표시 — 장음·촉음·ん 자동 감지
• 학습 기록 JSON 백업 — Settings → 백업 내보내기
• 라이트/다크 모드 지원

이 앱이 측정하는 것
• 한국어 뜻 인식 정답률 (첫 시도 / 회복 분리)

이 앱이 측정하지 않는 것
• 문맥규정·용법·읽기·발음 약점
