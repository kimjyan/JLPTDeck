# Security Policy

## Reporting a Vulnerability

If you discover a security issue in JLPTDeck, please report it privately rather than opening a public issue.

- Contact: open a private GitHub security advisory on this repository
- Expected response time: within 7 days

## Scope

- SwiftData local storage handling
- JMdict parsing and bundled data integrity
- Any network or update channels introduced later

## Out of Scope

- Issues requiring a jailbroken device
- Social engineering of maintainers
